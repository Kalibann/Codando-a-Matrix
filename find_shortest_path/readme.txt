Código em Python para percorrer um grafo através de uma lista de adjacência passada e 
então encontrar o menor caminho entre um vértice inicial e um vértice final. Após encontrá-lo,
plotar o grafo de acordo com os parâmetros passados (espaço de ocupação, cores, layout), 
utilizando a biblioteca igraph.

Mais sobre Teoria dos Grafos: https://www.ensinoeinformacao.com/teoria-dos-grafos
Mais sobre o igraph: https://igraph.org/python/
